# Pokemon List 

This a small project is intended as part of my personal portfolio. It connects to [The Unofficial GraphQL for PokeAPI](https://graphql-pokeapi.vercel.app/).

The following tech stack is used:
* React
* TypeScript
* GraphQL
* ApolloClient
* MaterialUI
